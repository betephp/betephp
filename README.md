# BetePHP

BetePHP是一个简单、易用的高性能PHP框架。我们坚信一个好的PHP框架应该是简单干净、易学易用，且不影响性能的。BetePHP将常用的组件功能使用统一的接口形式设计封装，无任何外部库依赖，实现了一个纯净、轻量级的高性能PHP框架。主要特点包括如下：

* 简单灵活的路由引擎
* 高性能
* 强大的组件库
* 统一的接口形式
